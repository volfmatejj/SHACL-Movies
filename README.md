# SHACL Movie Validation

This repository contains the technical files for a semester project focused on
SHACL validation of movie data from DBpedia.

The descriptive part of the project is submitted separately as a written report.

## Files in this Repository

- `shapes_movies_en.ttl` – SHACL shapes for validating `dbo:Film` entities
- `movies_en.ttl` – Movie data extracted from DBpedia
- `movies_en_synthetic_valid.ttl` – Synthetic dataset with valid movie records
- `movies_en_synthetic_errors.ttl` – Synthetic dataset with intentionally invalid data
- `SPARQL Query.txt` – SPARQL CONSTRUCT query used to extract data from DBpedia
- `Validation_Synthetic_Errors.txt` – Validation report for synthetic error data
- `Validation Report_Real data.txt` – Validation report for real-world DBpedia data
- `SHACL - Movies.docx` – Final written project report

## Tools Used

- DBpedia SPARQL endpoint
- SHACL Playground
